> Bad Bank three-tier application that uses React, Node.js/Express and MongoDB on a Docker container.

## Install

```sh
npm install
```

## Usage

```sh
docker run -p 27017:27017 -name badbank -d mongoDB
node index.js
```

## Run tests

```sh
npm run test
```
